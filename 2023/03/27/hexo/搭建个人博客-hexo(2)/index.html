快速搭建hexo个人博客(2)

2023/3/27 教程 hexo gitee github

本篇将介绍如何将个人博客部署到github或者gitee上面

一、 部署到gitee

1. 新建仓库

仓库的路径格式必须是 用户名.gitee.io才行
比如你的gitee注册的名字是xxx, 路径就为xxx.gitee.io

2. 安装hexo上传插件

$ npm install hexo-deployer-git --save

3. 修改hexo仓库路径

deploy:   
  type: git   
  repo: 仓库路径  
  branch: master

4. 推送仓库

$ hexo d

5. 开启Gitee Pages功能

gitee这边如果想使用其站点功能，需要开启Gitee Pages功能,开启这个功能需要进行实名认证，
根据引导完成认证后，即可开启该功能

开通成功之后可以通过https://xxx.gitee.io/xxx.gitee.io访问

二、 部署到github

和gitee步骤差不多，只是不需要开启Gitee Pages功能
推送仓库后通过https://xxx.github.io访问
