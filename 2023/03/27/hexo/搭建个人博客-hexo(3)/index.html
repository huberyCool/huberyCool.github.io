快速搭建hexo个人博客(3)

2023/3/27 hexo教程 hexo

本节将介绍一些编写hexo博客的常用指令

1. 创建一个新的博客页面

$ hexo new '新的博客'

执行命令后会在source文件夹下生成一个名为 <新的博客.md> 的文件
编写完成后可以发布到仓库

$ hexo cl # 清理
$ hexo g  # 生成静态页面
$ hexo d  # 发布到仓库

2. 创建草稿

$ hexo new draft d # 新建草稿文件
$ hexo server --draft # 预览草稿文件
$ hexo publish b # 发布草稿文件
