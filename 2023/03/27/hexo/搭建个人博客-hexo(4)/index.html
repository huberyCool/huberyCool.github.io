快速搭建hexo个人博客(4)

2023/3/27 教程 hexo

本节将记录一些常见的错误及其解决方法

一、 发布时报Spawn failed

解决方案

  1. 首先删除博客文件夹根目录下的.deploy_git/文件夹
  2. 执行git命令
$ git config --global core.autocrlf false
  1. 清理
$ hexo cl
  1. 生成静态文件
$ hexo g
  1. 发布
$ hexo d
