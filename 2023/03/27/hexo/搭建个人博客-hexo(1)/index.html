快速搭建hexo个人博客(1)

2023/3/27 hexo教程 hexo node.js git

本篇将介绍如何准备开发环境，如何进行博客本地预览

一、环境准备

1. 安装Node.js

直接官网下载 Node.js

由于众所周知的问题，可以安装淘宝源cnpm

$ npm install -g cnpm --registry=https://registry.npm.taobao.org

2. 安装git

官网下载Git

二、安装hexo

1. 安装hexo-cli

$ npm install -g hexo-cli

或

$ cnpm install -g hexo-cli

安装完成后查看hexo版本号,如果正常输出，视为安装成功

$ hexo -v

2. 初始化自建博客文件夹

$ hexo init your_blog_name

3. 安装依赖

$ cd your_blog_name && npm install

4. 生成网站文件

$ hexo g

5. 启动服务

$ hexo s

启动成功后，访问http://localhost:4000,即可进行本地访问
